### Overview
This folder contains multiple .dat files 

  *_wvnm_eps_raw.dat 

which contains tabulated material data from the default Lumerical material database. 
References for each data can be found from the below list, using the keyword in the file title Palik/CRC/JnC. 

######################################################## 
### File format
each line contains the following information.  
  wavelength(nm)  real(permittivity)  imag(permittivity)
